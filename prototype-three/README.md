# Prototype Three

Static scaffold for the full CX_lab Dojo advanced-containment demo.

Open `index.html` directly in a browser. No build step is required.

## Demo Flow

1. Persona: Maya R., loyal shopper / VIP parent.
2. 128 Experiments: strategy arms are generated and narrowed.
3. Live Call: policy-first baseline branch fails containment.
4. Score: Person 2 evaluation creates pruning evidence.
5. Dream Pass: offline cluster finds the reusable failure pattern.
6. Next Iteration: deadline-inventory-first branch improves containment.
